## 📝 Présentation globale du projet
Scam-Center est une ressource utilisable publiquement qui permet à n'importe qui de référencer, dans une base de données, des noms de domaines potentiellement malveillants. 

L'objectif est d'offrir un environnement numérique plus sûr contre l'ingénierie sociale. Par exemple, un lien suspect reçu par SMS peut être enregistré sur la plateforme afin que d'autres utilisateurs puissent vérifier sa fiabilité avant de cliquer. Le site permet de consulter les rapports détaillés laissés par la communauté pour apprendre à identifier les menaces.

## 👥 Équipe et Répartition du travail
Le projet a été développé en binôme avec une organisation claire des tâches :

* **Développement Backend & Base de données (Mon rôle) :** Réalisé par Augustin de Vivar. J'ai conçu la logique fonctionnelle en PHP (système d'alerte et de vérification), l'architecture de la base de données, ainsi que la mise en ligne du service.
* **Développement Frontend :** Réalisé par Matthieu Cabot. Il a conçu l'architecture HTML/CSS et l'identité visuelle de la plateforme, en optant pour un style "vintage" accessible.

## 🛠️ Technologies et Outils
* **Langages :** PHP pour le backend dynamique, HTML et CSS pour l'interface.
* **Environnement local :** Xampp pour la simulation des bases de données et du serveur PHP.
* **Versionnement :** Git et GitHub pour la gestion collaborative du code.

## 🚀 Pistes d'amélioration et d'ouverture
Lors de la conception pour les Trophées NSI, nous avons identifié plusieurs axes de développement futur :
* **Sécurité :** Protection contre les injections SQL pour sécuriser les données des utilisateurs.
* **Interactivité :** Système de commentaires et système de notation "pouce vert / rouge" pour valider la qualité des rapports.
* **Accessibilité :** Création d'une API pour faciliter la diffusion des données et améliorer la fiabilité globale.
* **Visibilité :** Optimisation du référencement (SEO) et affichage des derniers sites signalés sous forme d'articles.

## ⚙️ Utilisation
Le projet nécessite un environnement serveur supportant PHP et MySQL (tel que Xampp, Wamp ou un serveur LAMP) pour fonctionner correctement en local.
